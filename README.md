# Lesson Template Repository

このリポジトリは、授業で使用する **課題配布用テンプレート** です。  
学生が直接このリポジトリを操作することはありません。

---

## このリポジトリの役割

- 授業中に配布する **課題・指示・補足情報の配布元**
- GitHub Classroom を通じて、各自の課題リポジトリに内容が配布されます
- 授業途中の追加・修正は **Pull Request（PR）** として配信されます

---

## 授業中の基本ルール（重要）

### 1. 問題文について
- 課題の内容は `question.md` に記載されます
- 授業中に内容が追加・修正される場合があります

### 2. 課題の更新方法
- 課題内容が更新されると、**Pull Request が届きます**
- 必ず PR の内容（差分）を確認し、**自分でマージ**してください

> ※ マージしないと、最新の課題内容が反映されません

---

## 学生が作業する場所

学生が作業するのは **自分専用のリポジトリ** です。

例：00-warmup-<your-github-username>


このテンプレートリポジトリを直接編集することはありません。

---

## 提出の考え方

- 課題の提出は **GitHub へのコミット / PR 作成** によって行います
- 詳細な提出方法は、各回の `question.md` に従ってください

---

## ディレクトリ構成（例）

課題によって、次のような構成を使用することがあります。

.
├─ question.md # 問題文
├─ submission/ # 提出物を置く場所（指定がある場合）
└─ README.md # この説明


---

## 注意事項

- 指示のないファイル・ディレクトリを削除しないでください
- 他の学生のリポジトリを参照・コピーしないでください
- 分からないことがあれば、授業中または指示された方法で質問してください

---

## この仕組みの目的

この授業では、次の力を身につけることを目的としています。

- GitHub を使った **課題受け取り・提出**
- Pull Request を使った **指示の確認・反映**
- 実務に近い形での **開発フローの理解**

---

以上を理解したうえで、授業に参加してください。
